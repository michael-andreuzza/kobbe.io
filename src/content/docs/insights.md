The **Insights** page helps you spot patterns in traffic and conversions without leaving the main site navigation. Open it from the sidebar for any site.

## What you see

### Insight metrics

A KPI strip at the top summarizes the selected range. Metrics can include average daily visitors, single-visit share, custom event totals, and pageviews. When revenue attribution is connected, Kobbe also shows average daily revenue, revenue per visitor, and median time to purchase.

Use **Add comparison** in the toolbar to compare the active range against a previous period. KPI values and breakdown tables follow the same comparison window as the overview dashboard.

### Breakdown tables

Six tables rank what drove activity in the range:

| Table | What it shows |
| ----- | ------------- |
| **Top sources** | External referrers with pageview share (and revenue when available). |
| **Top countries** | Countries with pageview share. |
| **Devices** | Desktop, mobile, and tablet mix. |
| **Browsers** | Browser families with share. |
| **Top pages** | Paths with the most pageviews. |
| **Top converting events** | Custom events ranked by count. |

Each table includes a share column. Use the share control in a card header to copy or download a branded image of that breakdown.

### Conversion peak

The **Conversion peak** heatmap shows when custom events happen most often by day of week and hour of day. Darker cells mean more events in that bucket. Hover a cell for the exact count.

The heatmap uses custom events only. If you have not sent custom events in the range, the card stays empty.

## Time range

Insights respects the same range control as the overview dashboard. Presets, calendar windows, and custom dates all apply to KPIs, tables, and the heatmap.
