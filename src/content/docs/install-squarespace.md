Squarespace **Code Injection** is the right place for site-wide analytics. Token: [Add the tracker](/docs/add-the-tracker).

## Tracker snippet

```html
<script
  defer
  data-token="YOUR_SITE_TOKEN"
  src="https://app.kobbe.io/tracker.js"
></script>
```

## Steps

1. **Settings → Advanced → Code Injection**.
2. Paste into **Header**.
3. Save.

Some templates or plan tiers affect availability; Business / Commerce tiers typically include header injection.

## Verify

Visit your live domain and check Kobbe.
