Kobbe lets you download a CSV export for each site. Use exports when you want to keep a backup, share numbers with your team, or analyze traffic in a spreadsheet.

## Export a site

1. Open the site in Kobbe.
2. Go to **Settings**.
3. Find **Data export**.
4. Choose the range you want to download: **Last 7 days**, **Last 30 days**, **Last 90 days**, or **All time**.

Your browser downloads a CSV file for the selected site and time range.

## What is included

Exports include the analytics events collected for that site:

- Pageviews.
- Custom events.
- Page path.
- Referrer origin, when available.
- Event time.
- Sanitized event properties, when available.

The export uses the same privacy rules as the dashboard. Kobbe does not include cookies, persistent visitor identifiers, raw IP addresses, or full URLs with query strings.

## When to use exports

Use CSV exports for customer reports, internal analysis, long-term backups, or moving data into another tool.
